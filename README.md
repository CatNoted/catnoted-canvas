## What it is

catnoted canvas is a sub-project of catnoted. It fuses two ideas into a single, hyper-merged tool for creative work:

- **Notion-like structure:** rich text, database tables, to-do lists.
- **Miro-like freedom:** a free-form canvas where you place sticky notes and drawings anywhere.

All of it lives in one space. It is **local-first** (your data stays in your browser, works offline) and **privacy-focused** (no telemetry). Real-time collaboration and AI assist are planned as opt-in layers.
